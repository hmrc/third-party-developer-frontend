Add an application to the sandbox

To start using our RESTful APIs with your application, you'll need to:

  1. add it to our test environment (sandbox)
  2. choose which sandbox APIs you want to use
  3. get credentials your application needs to interact with our APIs
  4. test how your application integrates with our APIs
Warning Make sure your application complies with our terms of use

If you have not already done so:

  • find out which RESTful and XML APIs are offered by HMRC
  • read up on authorisation
