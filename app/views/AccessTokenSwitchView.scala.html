Switch to an OAuth 2.0 access token

Using application-restricted endpoints with server tokens is now deprecated.

To continue using your application, you need @{Link.toInternalPage(url = s"${appConfig.apiDocumentationFrontendUrl}/api-documentation/docs/authorisation/application-restricted-endpoints", value = Some("an OAuth 2.0 access token")).toHtml} to make API calls. OAuth 2.0 access tokens are secure and better protect your data.

Server token retires on:

  • 1 September 2020 in sandbox
  • 1 October 2020 in production

You must switch to an OAuth 2.0 access token before server token is retired. If you do nothing, your application may not work.
